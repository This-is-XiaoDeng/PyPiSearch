# PyPiSearch

## 简介

这是一个 PyPi 搜索程序，一定程度上解决了 pip 不支持搜包的问题

## 依赖

1. Python3
2. rich
3. requests 

## 使用方法

1. 安装 Python3
2. 安装依赖
```bash
pip install rich requests
```
3. 运行程序
```bash
python3 main.py
```
